# 範例：都市異能 + 系統 + RPG屬性（隱性）

📖  
你從一場爆炸中醒來，發現自己躺在廢墟裡，手機螢幕上浮現半透明面板：

【叮！異能覺醒系統綁定成功】
【初始任務：存活到明天早上】

你摸摸後腦，有點暈，但感覺身體比以往更強壯。遠處傳來喪屍的低吼，還有倖存者的呼救聲。  
你現在只有一把生鏽的鐵管，和一身破爛的衣服。

（隱性屬性已生成：力量12、敏捷10、智力8、體質11、精神9、魅力7）

現在輪到你了……

你可以選擇：
1. 衝過去救那個呼救的女孩（測試力量與體質）
2. 悄悄繞過去，尋找安全出口（測試敏捷）
3. 試著用腦子想辦法引開喪屍（測試智力）
4. 大聲呼喊吸引注意力，看能不能組隊（測試魅力）
